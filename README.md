3D-Socket
=========

three.jsのサンプルコードをSocket.ioでMMO化したものです。

複数ブラウザを立ち上げると、全てのキャラクターの動きが同期します。

<img src="https://github.com/naotaro0123/3D-Socket/blob/master/image.jpeg" width=550> 


## Build Setup

``` bash
# install dependencies
npm install

# express start
npm run start

# access browser
http://localhost:3001/
```
